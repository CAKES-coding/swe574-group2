## Installation information

### Customizable

1. Open docker-compose.yml file in a text editor.

2. Enter variables with braclets {{ }} with your own values. 

3. Save.

4. Enter below command.

`docker-compose up`

5. To stop container.

Control+C, then `docker-compose down`
